# Remont.Krd — Лендинг «Ремонт квартир»

Landing page for a fictional apartment renovation company in Krasnodar. Built as a portfolio project to demonstrate realistic business landing page structure, form handling, and scroll animations.

## Live Demo

[remont-krd on GitHub Pages](https://georgy-itech.github.io/appartment-recover/)

## Features

- Contact form with client-side validation
- Parallax effect on hero image
- Scroll-reveal animations via IntersectionObserver
- Work gallery with project cards
- Benefits section with icon grid
- Responsive layout from 375px up

## Tech Stack

- HTML5 (semantic, accessible)
- Pure CSS (custom properties, clamp(), BEM)
- Vanilla JavaScript (no libraries)

## Project Structure

```
appartment-recover/
  index.html
  styles.css
  script.js
  favicon.svg
  og-image.svg
```

## What I Practiced

- Real business landing page structure (hero → services → gallery → benefits → form)
- CSS Grid with multi-column breakpoints
- `IntersectionObserver` for scroll-reveal
- Form validation without `alert()`
- Parallax with `requestAnimationFrame`
